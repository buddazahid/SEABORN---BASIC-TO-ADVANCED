# Seaborn - Basic to Advanced

A simple Python script demonstrating how to use Seaborn for common statistical visualizations.

## Overview

This repository contains a single example script:

- `SEABORN - BASIC TO ADVANCED.py`

The script loads the built-in Seaborn `tips` dataset and creates several plots, including:

- Line plot
- Bar plot
- Scatter plot
- Histogram
- KDE plot
- Box plot
- Violin plot
- Count plot
- Heatmap
- Pair plot
- Joint plot
- Styled plot with Seaborn themes

## Requirements

- Python 3.8+ (recommended)
- seaborn
- matplotlib
- pandas
- numpy

## Installation

Install the required packages with pip:

```bash
pip install seaborn matplotlib pandas numpy
```

## Usage

Run the script from the command line:

```bash
python "SEABORN - BASIC TO ADVANCED.py"
```

Each plot will be displayed one at a time.

## Notes

- The script uses Seaborn's built-in `tips` dataset.
- Plot windows may need to be closed before the script continues to the next plot.
- This repository is intended as a learning aid for Seaborn visualization basics.
